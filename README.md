# Расчетно-графическая работа по дисциплине Машинное обучение и большие данные

## Тема. Разработка программного комплекса для инференса моделей машинного обучения и предиктивного экологического мониторинга

Программный комплекс представляет собой интерактивное веб-приложение на базе фреймворка Streamlit. Он разработан для анализа качества атмосферного воздуха и прогнозирования концентрации оксида углерода CO(GT) на основе многомерных физико-химических показателей. Научное исследование проведено на базе набора данных AirQualityUCI.

### Результаты оценки эффективности прогнозных моделей на тестовой выборке

В ходе выполнения работы были обучены, оптимизированы и сериализованы 6 разнородных архитектур моделей машинного обучения. Ниже приведены верифицированные статистические показатели их эффективности.

| Индекс модели | Архитектура прогнозного алгоритма | Коэффициент детерминации (R2) | Средняя абсолютная ошибка (MAE), мг/м³ |
| :--- | :--- | :---: | :---: |
| **ML5** | Stacking Regressor (Многоуровневый стекинг) | 0.9297 | 0.2381 |
| **ML3** | CatBoost Regressor (Симметричные деревья) | 0.9282 | 0.2463 |
| **ML4** | Random Forest Regressor (Ансамбль бэггинга) | 0.9214 | 0.2482 |
| **ML2** | Hist Gradient Boosting Regressor (Scikit-Learn) | 0.9200 | 0.2599 |
| **ML6** | Deep Neural Network (Полносвязная ИНС Keras) | 0.9114 | 0.2803 |
| **ML1** | Decision Tree Regressor (Решающее дерево) | 0.8449 | 0.3762 |

Ансамблевый метод стекинга (ML5) продемонстрировал наивысшую точность и обобщающую способность.

### Инструкция по развертыванию и локальному запуску

1. Клонирование удаленного репозитория проекта и переход в рабочий каталог.
```bash
git clone [https://github.com/ARKOD-PM/MLnDB_RGR.git](https://github.com/ARKOD-PM/MLnDB_RGR.git)
cd MLnDB_RGR
```

2. Инициализация и активация изолированного виртуального окружения venv.

```bash
python3 -m venv venv
source venv/bin/activate  # Для ОС Linux/macOS
# .\venv\Scripts\activate  # Для ОС Windows
```

3. Установка обязательных зависимостей и пакетов.

```bash
pip install -r requirements.txt
```

4. Инициализация процесса предварительного обучения и сериализации весов моделей.

```bash
python3 train_models.py
```

5. Запуск веб-сервера локального приложения Streamlit.

```bash
streamlit run app.py
```
